<h1 style="text-align:center;">Geolocation Map</h1>

## What is this?
A simple application to visualize your position on a map.

## How to use it?
- Run it!

```bash
$ npm run serve:commonjs:prod
```

- Build it!
```bash
$ npm run build:commonjs:prod
```
